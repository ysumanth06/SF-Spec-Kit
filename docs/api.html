Core CLI Commands

init add_box

Safe initialization of the `sfspeckit-data/` structure within a new project root.

$ sfspeckit init
branch account_tree

Creates standardized naming for `feat/`, `story/`, or `hotfix/` branches.

$ sfspeckit branch story/123
scaffold architecture

Generates the folder and documentation templates for a specific story or feature.

$ sfspeckit scaffold --feature=LWC-Grid
verify verified

Runs the full autonomous verification suite: Scanner + Tests + Evidence generation.

$ sfspeckit verify --strict
score speed

Calculates real-time project health and story completion metrics for architectural review.

$ sfspeckit score --summary
